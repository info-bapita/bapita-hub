Section breakdown — what's in each, what's optional

Always Hero Shop photo (full-bleed), name, tagline, "Book Appointment" CTA. Animation: name fades up on load, photo slow Ken Burns zoom. If no barber photo → stock photo matching template mood.
Always Services All active services from DB — name, duration, price, "Book →" per service. Clicking opens booking overlay pre-filled with that service. Animation: cards stagger fade-up on scroll.
Optional Gallery 1–3 photos from `gallery_images[]`. Layout: 1 large + 2 small. Missing photos filled with matching Unsplash stock. Toggle: `show_gallery`.
Optional About Short paragraph from `about_text` — pulled from barber's Instagram bio or Google Maps during onboarding. Toggle: `show_about`.
Optional Hours Auto-generated from `business_hours` JSONB (same data the slot API uses). No extra input needed. Toggle: `show_hours`.
Optional Location Address from `businesses.address` + "Get Directions →" → Google Maps. Toggle: `show_location`.
Always Floating CTA Sticky bottom bar, appears after scrolling past hero. "Shop Name · Book Appointment →". Opens booking overlay. Per-template color.
